# Projeto de Estudos em Programação

Criei este repositório com o objetivo de aprender e evoluir na programação.

Aqui vou reunir códigos, exercícios e pequenos projetos que desenvolvo ao longo dos estudos. A ideia é começar pelo básico e ir avançando aos poucos, trabalhando desde conceitos simples até simulações de projetos mais próximos da prática.

## Objetivo

- Melhorar a lógica de programação  
- Praticar conceitos fundamentais e mais avançados  
- Simular situações reais de desenvolvimento  

## Conteúdo

- Códigos simples  
- Exercícios de lógica  
- Projetos intermediários  
- Projetos mais completos ao longo do tempo  

## Observação

Este repositório faz parte do meu processo de aprendizado e evolução como desenvolvedor.
